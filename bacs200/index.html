Welcome to BACS 200

This website contains the content for a course offered at University of Northern Colorado. BACS 200 - Introduction to Web Development for Small Business.

Overview

The course is built around

  • 45 Lessons
  • 14 Student Projects

Lessons

Each lesson presents detailed information about a specific topic. The notes match the content of the recorded lectures.

Projects

Each project has detailed instructions about a requirements for success. The projects are assigned weekly to apply the knowledge that is taught in the lessons.
