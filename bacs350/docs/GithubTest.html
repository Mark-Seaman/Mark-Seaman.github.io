Test Github Pages

Here is a quick way to test your setup for BACS 200.

For this to work you must have completed the steps in the following documents.

  • Register a Github account
  • Create a Github repository
  • Publish Github Pages

Student Settings

Test your links. These links are automatically formed from the information that you configured for the "Github" setting in your student profile. If you discover an error then keep working until the proper pages load.

  • Github User: {{ github_user }}
  • Github URL: {{ github_url }}
  • Github Pages: {{ github_pages }}
  • Github Server: {{ github_server }}
  • Test Page: {{ github_server }}/index.html

Teacher Settings

If you have difficulties try to compare your settings against the teacher settings.

  • Github User: {{ teacher_github_user }}
  • Github URL: {{ teacher_github_url }}
  • Github Pages: {{ teacher_github_pages }}
  • Github Server: {{ teacher_github_server }}
  • Test Page: {{ teacher_github_server }}/index.html
