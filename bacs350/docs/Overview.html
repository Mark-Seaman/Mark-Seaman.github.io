Web Hosting

Github Pages

Github is a version control webservice that manages code repos for your projects. Github Pages allows you to automatically publish your web page content just by committing your code changes automatically.

Github Pages is free and reasonably easy to set up. Follow the instructions posted here to Setup Github Pages.

Github Version Control

Sign up for a Github Account to manage all of your code.

Github Desktop

Github Desktop is an application program that will let you work with your code repositories without having to visit the Github Website.
