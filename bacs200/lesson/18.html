Text Properties

  • font-family
  • font-size
  • font-weight
  • text-align
  • text-shadow

font-family

  • Inherited by all descendants
body {
    font-family: Arial, Helvetica, sans-serif;
}

p {
    font-family: "Times New Roman", Times, serif;
}

Size Units

  • Units of size for values
20px - 20 pixels
10pt - 10 points
2em - twice the line height (M)
50% - half the size

font-size

  • Multiple units
p {
    font-size: 20px;  /* in pixels */
    font-size: 12pt;  /* in points */
    font-size: 150%;  /* parent element */
    font-size: 1.5em;  /* M is line height */
}

font-weight

  • Special font treatment
p {
    font-weight: 400;  /* normal */
    font-weight: normal;  /* 400 */

    font-weight: 700;  /* bold */
    font-weight: bold;  /* 700 */

    font-weight: lighter;  /* parent element */
}

text-align

  • Where does the space go?
p {
    text-align: left;
    text-align: center;
    text-align: right;
}

text-shadow

  • Give a 3D look
h1 {
    text-shadow: 2px 2px;
}

h2 {
    text-shadow: -2px -2px 4px yellow;
}

font

  • Shorthand font property
p {
    font: italic bold 16px Arial, sans-serif;
    font: 120% Times;
}

Color Properties

  • color
  • background-color

Color Values

  • By Name
  • By RGB
  • By Hexidecimal
h1 {
    color: rgb(20%, 20%, 20%);  /* Red Green Blue */
    
    color: silver;              /* Named color */
    
    color: #0a0a0a;             /* Hex - Red Green Blue */
}

Hexidecimal Values

#01ffa5   -- Hex value

R -- 01 (1)
G -- ff (255)
B -- a5 (165)

Hexidecimal Colors

#ffffff  -- white
#000000  -- black
#080808  -- gray

#ff0000  -- red
#00ff00  -- green
#0000ff  -- blue
