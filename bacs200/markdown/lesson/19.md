### CSS Property Groups
* Text
* Color
* Spacing

### Spacing Properties
* margin
* padding
* width
* height


### Size of Elements
* width
* height

```css
h1 {
    width: 500px;
    height: 100px;
    border: solid 1px gray;
}
```


### Box Model
* Margin is outside
* Padding is inside

```css
body {
    padding: 50px;
    margin: 10px;
    border: solid 1px gray;
}
```


### Example Rule Set

```css
h1 {
    margin: 100px;
    background-color: rgb(20%, 20%, 20%);
}


body {
    padding: 200px;
    border: solid 2px yellow;
}
```


### Borders Properties
* border
* border-radius
* box-shadow

Examples

```css
h1 {
    border: solid 1px gray;
    border-radius: 20px;
    box-shadow: 2px 2px 5px gray;
}

```
